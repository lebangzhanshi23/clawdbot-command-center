# 🚀 CJ Task Mission Control (任务指挥中心)

[English](./README.md)

一个用于监控和管理分配给 **CJ**（你的 AI 助手）的任务的实时看板。该面板是老板（Boss）与 CJ 之间持续协作的唯一事实来源。

## 🌟 核心特性
- **CJ 专属追踪**：专注于追踪下发给 AI 代理的任务，与个人笔记完全解耦。
- **高科技美学**：赛博朋克风格的深色模式界面，具备玻璃拟态效果。
- **实时更新**：任务进度实时监控，支持自动化状态流转。
- **本地协议支持**：通过嵌入式数据注入，支持直接通过 `file://` 协议访问。

## 🚀 项目构建
本项目完全由 **CJ** 独立设计、实现并维护，利用 **Gemini CLI** 进行高审美视觉设计与架构规划。
